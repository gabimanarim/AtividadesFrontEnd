# Lista CAT Facts

Projeto simples em HTML, CSS, JavaScript para consumir uma API pública usando fetch.

## Objetivo

Exibir fatos aleatórios sobre gatos utilizando uma API externa e mostrar os dados em uma tabela.

## Tecnologias utilizadas

- HTML
- CSS
- JS
- Fetch API
- API pública: http://catfact.ninja

## Funcionalidades

- Botão para buscar fatos sobre gatos
- Consumo de API com fetch()
- Exibição dos dados em tabela
- Contador de registros
- Tratamento básico de erro no console

## Como executar

1. Abrir o arquivo index.html no navegador
2. Clicar no botão "Carregar Fatos sobre Gatos"
3. Os dados serão exibidos na tabela

## Observação

Projeto desenvolvido, em conjunto com o professor, para prática de consumo de API com JavaScript.
