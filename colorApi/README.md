# Atividade Color API

Projeto desenvolvido para a disciplina de FrontEnd;

# Objetivo
Criar uma aplicação em HTML, CSS e JavaScript que consome uma API pública utilizando fetch(), exibindo os dados em tabela e mostrando detalhes do item selecionado;

# Tecnologias utilizadas
HTML;
CSS;
JS;
Fetch();
Color API (https://www.thecolorapi.com);

# Funcionalidades
Busca de cor pelo código HEX;
Consumo de API usando fetch();
Exibição em tabela;
Background da linha com a cor retornada pela API;
Detalhe da cor ao clicar na linha;
Tratamento de erro;
Validação do input;
Botão para limpar tabela;
Loading durante requisição;
